# 🌐 Fundamental Web Programming

รวมแลปวิชา Fundamental Web Programming ซึ่งครอบคลุมเนื้อหาตั้งแต่ส่วน Front-end ไปจนถึง Back-end และ Database

## 🛠 เทคโนโลยีที่ใช้ (Tech Stack)
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, REST API, Session Management
* **Database:** SQLite3

## 📂 โครงสร้างโฟลเดอร์ (Folder Structure)
รายละเอียดของงานในแต่ละแฟ้ม:

* `lab2/` ถึง `lab11/` : โค้ดแบบฝึกหัดและงานของแต่ละสัปดาห์
* `test/` : โฟลเดอร์สำหรับการทดสอบโค้ดและการทดลองต่างๆ
