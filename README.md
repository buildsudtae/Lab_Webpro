# 🌐 Fundamental Web Programming

A collection of labs for the Fundamental Web Programming course, covering Front-end and Back-end development to Database management.

## 🛠 Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, REST API, Session Management
* **Database:** SQLite3

## 📂 Folder Structure
Details of each directory:

* `lab2/` to `lab11/`: Weekly lab exercises and assignments.
* `test/`: A directory for testing code and various experiments.
